# Enterprise Onboarding Checklist

- Domain verification and SSO configured (OAuth/SAML)
- MFA rules enforced for admins
- IP allowlists set for admin endpoints
- Create admin and ops roles; test least privilege
- Import users/contractors; map company_alias and master_account_id
- Data migration plan (jobs, timesheets, storage) with dry-run
- Configure backups, SLO alerts, and audit_logs export
- Enable feature flags for enterprise features (SSO, exports, approvals)
- Run smoke tests; sign off with customer
- Schedule quarterly access reviews and DR drill
