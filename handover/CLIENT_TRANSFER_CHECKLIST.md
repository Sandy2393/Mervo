# Client Transfer Checklist (APP_ID)

- Repo access: transfer ownership; archive access list.
- Cloud projects: hand over org/project ownership; billing account transfer.
- Domain: transfer registrar credentials; update contact info; confirm DNS.
- Supabase/DB: grant owner role; rotate admin passwords.
- Admin accounts: create client admin, enforce MFA.
- Runbooks: provide launch, rollback, migration, monitoring docs.
- Secrets: rotate all keys; share via vault (no email).
- Final invoices: deliver and confirm receipt.
- Signoff: client confirms access and functionality.
