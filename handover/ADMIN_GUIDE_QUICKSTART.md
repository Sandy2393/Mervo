# Admin Quickstart (APP_ID)

1) Login: go to https://app.example.com/admin; use provided admin account; enable MFA.
2) Add users: invite via email; set roles (admin/manager/worker).
3) Post jobs: create job with location/time; assign contractors.
4) Approve reports: review photo + notes; approve/reject.
5) Export data: download CSV reports; sync to payroll via exports.
6) Retention: set retention policy for photos/reports; default 90d.
7) Support: email support@APP_ID or open in-app ticket.
