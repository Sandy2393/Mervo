# Security Signoff Checklist (APP_ID)

- Secrets audit complete; no plaintext secrets in repo; vault paths verified.
- RLS policies enforced on all customer data tables; reviewed by security.
- Pen-test planned/executed; critical/high issues closed or accepted by exception.
- Dependency scan (SCA) run; zero criticals; highs documented with plan.
- SSO + MFA enforced for admins; audit logging enabled.
- Data residency checked; backups stored in approved regions.
- Privacy policy updated; TODO: legal review.
- SOC2 readiness notes compiled; gap list documented.
- Incident response tested; on-call coverage confirmed.
