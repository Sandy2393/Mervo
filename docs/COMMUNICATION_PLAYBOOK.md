# Communication Playbook

## Maintenance Window Template
Subject: [Notice] Scheduled Maintenance on <DATE/TIME> UTC
Body:
- What: Brief description
- Impact: Expected downtime or latency
- Window: Start/End
- Actions: None / what to do
- Contact: support@mervo.app

## Incident Template
Subject: [Incident] <SERVICE> Degraded
Body:
- Summary: What happened
- Impacted users/regions
- Start time (UTC)
- Mitigation steps underway
- Next update time
- Contact and status page link

## Feature Release Template
Subject: New in Mervo: <Feature>
Body:
- What shipped (1-2 bullets)
- Why it matters
- How to use (link to docs)
- Rollout plan and flags
- Feedback link (to FeedbackDashboard)

## Channels
- P1/P2: Pager + Slack #incidents
- P3: Slack + email
- P4: Release notes, status page

## Cadence
- Weekly release notes digest
- Monthly roadmap update
- Incident follow-ups within 48h of resolution
