# Billing Runbook (APP_ID)

- Setup billing account; assign owner; add payment methods.
- Transfer billing ownership to client at handover.
- Invoice generation: monthly cycle; export CSV using template; include tax where applicable.
- Refunds/disputes: evaluate case; process via payment provider; record credit memos.
- Reconciliation: match payouts vs invoices; use json2csv exports; review failed payments.
- Alerts: set budget alerts; use billing monitor script with webhook (placeholder).
