# Dashboard Spec

## Executive Dashboard
- KPIs: MAU, WAU, Jobs/week, Revenue.
- Filters: date range, company.
- Refresh: daily 06:00 UTC.
- Owner: Data team.

## Operations Dashboard
- KPIs: Pending approvals, Overdue jobs.
- Filters: company, region.
- Refresh: hourly.
- Owner: Ops.

## CS Dashboard
- KPIs: Onboarding funnel completion, Active tickets, Health alerts.
- Refresh: daily.
- Owner: CS.

## Finance Dashboard
- KPIs: Revenue vs payouts, AR aging, Refunds.
- Refresh: daily.
- Owner: Finance.
